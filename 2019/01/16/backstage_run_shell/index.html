服务器后台运行脚本

  1. 安装 yum install screen
  2. 打开新会话并起个名字：screen -S myname
  3. 执行脚本
  4. Ctrl + d 离开当前screen
  5. 回到主shell 此时可以关闭
  6. 在主shell里面执行screen -ls 查看当前会话列表
  7. screen -r myname 切换到执行脚本的shell
  8. 结束会话列表 exit
