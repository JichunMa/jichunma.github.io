Hexo新建文章并发布

  1. 进入博客目录,在/source/_posts文件夹下直接建立一个xx.md文件
  2. hexo generate #生成更改
  3. hexo server #本地预览生成后的页面
  4. hexo deploy #将生成的更改部署到GitHub上(自己域名下展示有些许延迟，可以先去GitHub上看代码是否成功)
